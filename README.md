# 电动汽车充电站布局优化项目
## 项目概述
本项目采用遗传算法（GA）和非支配排序遗传算法（NSGA-II）对电动汽车充电站进行多目标优化布局。项目综合考虑经济成本和用户满意度两个目标，为充电站选址和充电桩配置提供科学决策支持。

## 主要功能
数据预处理：读取充电需求和候选站点数据，进行网格化聚合处理

## 多目标优化：

* 使用NSGA-II算法寻找经济成本与用户满意度的Pareto前沿解

* 使用单目标遗传算法寻找综合最优解

## 可视化输出：

* 绘制Pareto前沿分布图

* 保存详细的优化结果到文本文件
